# Tixly

**Tixly** è un’app mobile innovativa pensata per gli amanti degli eventi. Offre un wallet digitale sicuro e privato per salvare e gestire biglietti di concerti, spettacoli, festival e altri eventi, con funzionalità social e diario personale per rivivere le esperienze.

---

## 🚀 Caratteristiche principali

- Wallet digitale sicuro per biglietti con caricamento foto e PDF  
- Reminder automatici per non dimenticare gli eventi  
- Diario multimediale per salvare foto, video, note e valutazioni post-evento  
- Feed social per condividere esperienze, chiedere info e interagire con la community  
- Sistema di convalida biglietti tramite QR/NFC (in sviluppo)  
- Interfaccia intuitiva e ottimizzata per giovani utenti (15-30 anni)  

---

## 📱 MVP - Funzionalità Base

- Login e autenticazione con Firebase Auth  
- Dashboard biglietti salvati con lista e dettaglio  
- Aggiunta/modifica/eliminazione biglietti (foto, nome, data, luogo, note)  
- Reminder evento con notifiche locali  
- Sezione diario personale con contenuti multimediali legati agli eventi  
- Persistenza dati tramite Firebase Firestore e Storage
